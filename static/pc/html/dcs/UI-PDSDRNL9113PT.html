꼭 알아두세요
  • 2010년 10월 1일부터 확정형 예정이율 상품의 보험계약대출이율이 기존 구간금리 방식에서 가산금리 방식으로 변경되어 가산금리 2.85% 이하의 보험계약대출이율은 변경 전 가산금리를 유지합니다.
  • 기준이율은 공시이율, 정기예금이율 상품별 적용이율입니다.
  • 최저보증이율이 있는 연동형 상품은 적용이율이 최저보증이율보다 낮을 경우 보험계약대출이율을 ‘최저보증이율 + 1.5%’로 적용합니다.
  • 보험계약대출이율 연동 상품의 보험계약대출이율은 2001년 7월 10일부터 ‘슈퍼재테크(배당형) 공시이율 + 1.5%’를 적용합니다.
