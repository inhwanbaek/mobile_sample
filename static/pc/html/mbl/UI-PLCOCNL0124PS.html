[PC → 휴대폰 복사]를 눌러 복사할 인증서를 선택하고
PC화면에서 생성된 인증번호를 교보생명 앱 화면에 입력하세요.

인증서 복사 자세히 알아보기

PC에 저장된 공동인증서를 휴대폰으로 복사하면 PC와 모바일에서 동일한 인증서로 모든 금융거래가 가능합니다.
  1. 1PC 홈페이지에서 [인증센터 > 공동인증서 > 인증서 내보내기] 메뉴를 선택하고 [PC→휴대폰 복사] 버튼을 눌러주세요.
  2. 2복사할 인증서를 선택하고 인증서 비밀번호를 입력해 주세요.
  3. 3교보생명 앱 실행 후 [인증센터 > 공동인증서 > PC에서 인증서 가져오기] 메뉴를 선택해 주세요.
  4. 4PC화면에서 생성된 인증번호를 교보생명 앱 화면에 입력해 주세요.
  5. 5교보생명 앱에서 인증서 비밀번호를 입력하면 공동인증서가 교보생명 앱으로 복사됩니다.
