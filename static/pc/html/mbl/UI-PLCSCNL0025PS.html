보안프로그램 오류해결

이용 중 오류 발생 시 삭제 후 재설치하거나 재실행하시기 바랍니다.

보안프로그램 설치 시 오류가 발생하여 ‘MY교보(나의 계약 서비스)’를 이용할 수 없는 경우 아래 내용을 참고하여 조치해 주시기 바랍니다.

오류 발생 시 여기로 문의하세요.

오류 발생 시 문의 전화번호 안내
교보생명 콜센터 1588-1001 키보드 보안프로그램 라온시큐어 1644-4128
개인PC방화벽 ㈜안랩 1577-9451 공동인증 프로그램 이니텍㈜ 1644-5040
  • 문의시간: 평일 09:00~18:00(토, 일, 공휴일 휴무)
  • 설치했던 프로그램을 삭제 후 재설치
  • 브라우저 캐시 삭제 후 재실행
Internet Explorer
01. 브라우저 상단에서 [메뉴 > 도구 > 인터넷옵션] 메뉴를 선택하여 검색 기록의 [삭제] 버튼을 클릭합니다.
02. 검색 기록 삭제 창에서 [삭제] 버튼을 클릭하여 브라우저 캐시를 제거합니다.
크롬
01. 주소창에 ‘chrome://chrome/settings/’ 라고 입력 후 나오는 설정 화면에서 고급 설정 표시를 클릭합니다.
02. [인터넷 사용 정보 삭제] 버튼을 클릭합니다.
03. 최근 방문 기록 삭제 창에서 삭제할 기간을 ‘전체’로 설정하고 삭제합니다.
사파리
01. ctrl +, 단축키를 눌러 기본설정 화면을 호출한 후 개인 정보 탭을 누르고 [모든 웹 사이트 데이터 제거]를 클릭합니다.
02. 제거 확인 창이 뜨면 [지금 제거] 버튼을 클릭해 브라우저 캐시를 삭제합니다.
파이어폭스
01. 브라우저 상단에서 [메뉴 > 도구 > 설정 메뉴]를 선택하여 개인 정보 탭을 클릭합니다.(메뉴가 보이지 않을 경우 Alt키를 눌러 메뉴를 활성화 합니다.) 개인 정보 탭 화면에서 최근 방문 기록을 삭제합니다.
02. 최근 방문 기록 삭제 창에서 삭제할 기간을 ‘전체’로 설정하고 삭제합니다.
오페라
01. Ctrl키와 F12 단축키를 같이 눌러 환경설정 화면을 호출한 후 고급 설정 탭을 클릭합니다.
02. 고급 설정 탭에서 방문목록을 선택하고 [지금 비우기] 버튼을 클릭합니다.
