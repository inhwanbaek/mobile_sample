보험계약 해지(해약) 신청

해지환급금을 수령할 계좌를 확인해 주세요.

해지환급금 수령액 확인

  • 총 신청건수
    20건
  • 지급금 합계
    1,000,000원
  • 차감지급금액(환급율) 합계
    1,000,000원
건별 상세내역

계좌선택

꼭 알아두세요
  • 실수령액 = (해지기본금 + 배당금 + 기타지급금) – (보험계약대출원금 + 보험계약대출이자 + 기타공제액)
  • 해지환급금을 수령한 후에는 취소할 수 없으므로 선택하신 보험증권번호를 다시 확인 후 신청하시기 바랍니다.
  • 해지(해약) 신청과 동시에 보험계약의 효력이 소멸되며, 이후 보험사고 발생 시 보장되지 않습니다. 보험계약을 한 번 해지하면 취소할 수 없으므로 해지하려는 계약의 보험증권번호를 정확하게 선택하시기 바랍니다.
