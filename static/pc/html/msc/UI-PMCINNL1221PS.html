추가납입보험료 자동이체 안내
  • 보험료가 해당 이체일 은행 영업시간까지 계좌에 입금되어 있어야 자동이체됩니다.
  • 계좌 잔액 부족, 지급제한, 약정대출금 연체 등으로 대체납부 불가능한 경우 자동이체가 실행되지 않습니다.
  • 추가납입보험료 자동이체 신청 시 매월 지정한 이체희망일에 1회만 청구 및 자동이체됩니다.
  • 지정된 이체희망일에 이체되지 않아 다음 이체일에 이체되는 경우 기본보험료만 자동이체되고, 추가납입보험료는 청구되지 않습니다.
