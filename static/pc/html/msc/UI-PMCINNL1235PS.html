UL자유납입보험료 자동이체납입제도란?

매월 납입해야 하는 유니버셜상품의 자유납입보험료를 고객이 지정한 은행 입출금계좌에서 지정한 이체일에 출금하여 자동으로 납입하는 제도입니다.

유의사항

  • 최초 이체시작일: 최초 이체시작일은 신청일로부터 유효합니다. 자동이체 등록일이 이체희망일 이전이면 신청하신 달부터 최초 이체가 시작되고, 자동이체 등록일이 이체희망일 이후면 신청한 다음 달부터 이체가 시작됩니다.
  • UL자유납입보험료는 이체희망일에 보험료가 이체되지 않을 경우 해당월 말일까지 5일 단위로 재청구됩니다.
  • UL자유납입보험료 자동이체 신청은 유니버셜 기능이 부가되고 보험료 납입의무기간이 지난 계약만 가능합니다. 유니버셜 기능 부가 상품의 보험료 납입의무기간은 가입상품별, 가입날짜별로 다릅니다.(2년, 3년, 4년, 5년, 7년, 10년, 12년 등)
  • 보험료가 해당 이체일 은행 영업시간까지 입금되어 있어야 자동이체됩니다.
  • 계좌 잔액 부족, 지급제한, 약정대출 연체 등으로 대체납부 불가능한 경우 자동이체되지 않습니다.
