  • 변액보험 계약의 펀드변경은 매년(계약일 기준) 총 12회 이내에서 가능합니다.
  • 각 펀드비율의 합이 100%이면 펀드변경이 가능합니다.
  • 펀드자동재배분 옵션을 선택한 계약은 적립금/기본보험료/추가납입보험료의 펀드변경 시, 같은 펀드를 선택해야만 변경이 가능합니다.
  • 평균분할투자(DCA) 옵션을 선택한 계약은 옵션을 [선택안함]으로 변경해야 펀드변경이 가능합니다.
  • 교육자금 펀드는 펀드변경 시 1개의 펀드만 선택할 수 있습니다.(아이미래 상품인 경우)
