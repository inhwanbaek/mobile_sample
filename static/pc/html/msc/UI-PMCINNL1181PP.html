보험료 자동이체납입제도 안내

보험료 자동이체납입제도란?

매월 납입해야 하는 보험료를 고객이 지정한 은행 입출금계좌에서 지정한 이체일에 출금하여 자동으로 납입하는 제도입니다.

유의사항

  • 보험료가 해당 이체일 은행 영업시간까지 계좌에 입금되어 있어야만 자동이체됩니다.
  • 계좌 잔액 부족, 지급제한, 약정대출금 연체 등으로 대체납부가 불가능한 경우 자동이체가 실행되지 않습니다.
  • 지정한 이체일에 출금되지 않은 경우 보험료 납입최고(독촉) 기간 동안 5일 단위로 자동이체가 다시 실행됩니다.
  • 기본보험료 자동이체일이 5일, 10일, 15일, 20일, 25일인 경우에만 추가납입보험료 자동이체를 신청할 수 있습니다.
