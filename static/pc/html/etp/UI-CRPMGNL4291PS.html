  • 첨부 가능한 파일 형식은 ‘xls’ 또는 ‘xlsx’입니다.
꼭 알아두세요
  • 가입자 일괄 업로드 절차
    • 업로드 전 적립비율 재산출 여부를 선택하시기 바랍니다. 선택하지 않으면 적립비율이 재산출되지 않습니다.
    • 적립비율은 파일 업로드가 완료된 다음 날 09:00에 처리됩니다.
    • 퇴직급여추계액 산출 기준일: 추계액을 산출하여 적용 및 변경하는 기준일
    • 적립비율 산출 기준일: 추계액이 변경됨에 따라 적립비율 산출 시 적용되는 적립금 기준일
    • 일괄등록 양식을 다운로드해서 가입자를 추가 또는 변경하여 첨부하면 정상 건은 처리되고 오류 건은 따로 수정하실 수 있습니다.
    • 첨부할 파일 선택 후 [파일첨부 및 검증]을 누르면 데이터 검증 후 업로드가 완료됩니다.
    • 업로드가 완료되면 해당 화면에서 업로드 결과를 확인하고, 오류 데이터를 정정하거나 다시 파일을 다운로드할 수 있습니다.
  • 데이터 입력 시 주의사항
    • 데이터를 입력하지 않은 항목은 표시되지 않습니다. 항목에 ‘0원’을 표시하려면 숫자 ‘0’을 입력하시기 바랍니다.
    • 평균임금의 변경범위는 전년도 기준 금액의 70~130%로 제한되어 있으며 이 범위를 벗어나면 오류로 처리됩니다.
    • 정보 등록 시, 주민등록번호 기준으로 이미 등록되어 있는 가입자라면 기존 정보를 ‘변경’으로 처리하고, 등록되지 않은 가입자라면 ‘신규 가입자’로 처리합니다.
