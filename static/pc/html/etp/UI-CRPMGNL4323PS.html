기본부담금 외 퇴직정산 부담금
퇴직정산 부담금은 퇴사자의 부담금을 마지막으로 납입한 날부터 퇴직 처리 전까지 기간에 대한 부담금입니다.
또는 경영성과급
경영성과급은 납입대상으로 지정된 가입자만 납입하는 부담금입니다. 납입대상 단체에게만 [경영성과급 납입] 버튼이 보입니다.
을 납입하시려면 아래 해당 버튼을 눌러 주세요.
꼭 알아두세요
  • 신규 가입자 또는 퇴직자가 있는 경우에는 신규 가입이나 퇴직 처리를 먼저 해야 합니다.
  • 신규 가입자를 추가하면 추가 즉시 정기부담금 납입대상 목록에 추가됩니다.(미납부담금은 예외)
  • 가입자가 100명 이상인 경우 일괄납입으로 자동 연결됩니다.

정기부담금 납입대상이 없습니다.

귀사의 퇴직연금 규약상 납입주기에 따라 계산된 다음 부담금 납입일은 2021년 3월 22일입니다.

자동 생성 전에 미리 납입하시려면 [정기부담금 미리 납입]을 눌러 주세요.
