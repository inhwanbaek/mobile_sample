  • 파일첨부
  • 임시 등록내역

파일첨부

파일 선택
  • 「근로자퇴직급여보장법」에 따라 가입자교육 및 퇴직연금운용현황을 가입자에게 알리기 위해 가입자의 연락처(휴대폰번호, 이메일, 집주소)를 꼭 입력해 주세요.
  • 첨부 가능한 파일 형식은 ‘xls’ 또는 ‘xlsx’입니다.
꼭 알아두세요
  • 가입자 일괄 업로드 절차
    • 일괄등록 양식을 다운로드해서 가입자를 추가 또는 변경하여 첨부하면 정상 건은 처리되고 오류 건은 따로 수정하실 수 있습니다.
    • 첨부할 파일 선택 후 [파일첨부 및 검증]을 누르면 데이터 검증 후 업로드가 완료됩니다.
    • 업로드가 완료되면 해당 화면에서 업로드 결과를 확인하고, 오류 데이터를 정정하거나 다시 파일을 다운로드할 수 있습니다.
  • 데이터 입력 시 주의사항
    • 데이터를 입력하지 않은 항목은 표시되지 않습니다. 항목에 ‘0원’을 표시하려면 숫자 ‘0’을 입력하시기 바랍니다.
    • 주민등록번호 기준으로 이미 등록되어 있는 가입자라면 기존 정보를 ‘변경’으로 처리하고, 등록되지 않은 가입자라면 ‘신규 가입자’로 처리합니다.
  • 가입자 연락처 필수 입력 안내
    • 「근로자퇴직급여보장법」에 따라 가입자교육 및 퇴직연금운용현황을 가입자에게 통지하기 위해 가입자의 연락처(휴대폰번호, 이메일, 집주소)를 꼭 입력해 주세요.
