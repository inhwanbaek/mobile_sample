1. 부담금 납입대상 지연이자 발생

법정휴직으로 부담금을 반드시 납입해야 함에도 불구하고 납입하지 않은 경우.

예) 육아휴직, 출산전후휴가, 산재휴업기간, 사용자귀책휴업, 수습기간, 적법한 쟁의기간, 사용자의 승인을 받은 휴업기간

2. 부담금 납입 미대상 지연이자 발생안함

법정휴직을 제외하고 별도로 사규에서 ‘휴직을 근속기간에 포함하지 않을 것’을 명시한 경우.

예) “회사의 휴직 중 자기계발 휴직은 근속기간에 포함하지 않기로 정한다.” 등의 사규에 해당하는 경우

3. 육아휴직(복귀 시 일시 납입)지연이자 발생안함

일시에 납부하기로 사규에 명시한 경우.

단, 육아휴직기간의 부담금을 업무복귀 시점에 일시에 납부하기로 정했음에도 불구하고 복귀 시점에 납부하지 않은 경우에는 육아휴직기간 동안의 부담금과 지연이자를 함께 납입해야 함.

파일첨부

  • 이번 납입대상이 아닌 가입자는 미납사유를 반드시 숫자(1, 2, 3 중 하나)로 입력하여 저장한 후 해당 파일을 첨부해 주세요.

꼭 알아두세요
  • 신규 가입자 또는 퇴직자가 있는 경우에는 신규 가입이나 퇴직 처리를 먼저 해야 합니다.
  • 신규 가입자를 추가하면 추가 즉시 정기부담금 납입대상 목록에 추가됩니다.(미납부담금은 예외)

  • 입력값의 셀 서식은 ‘일반’ 또는 ‘텍스트’로 설정되어 있으므로, 임의로 변경하면 데이터 처리에 오류가 발생합니다.
