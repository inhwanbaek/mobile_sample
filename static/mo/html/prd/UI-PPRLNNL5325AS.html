출금계좌

이자 자동이체일

꼭 확인하세요
  • 보험계약대출 원리금 청구 중인 계약으로 보험계약대출을 받을 경우 원리금이 이충출금될 수 있습니다. 이중출금된 이자는 2일(영업일 기준) 안에 다시 입금해 드립니다.
  • 매월 발생하는 보험계약대출 이자는 은행 자동이체로 납부하셔야 합니다.
  • 이자는 보험계약대출 신청 시 등록한 입출금계좌에서 지정한 자동이체일에 출금됩니다.
  • 원리금 분할상환을 계속 이용하시려면 추가대출 후 대출원리금 분할상환을 재신청해 주시기 바랍니다.
