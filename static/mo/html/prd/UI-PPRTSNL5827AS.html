안내사항

  • 예약한 시간 10분 전에 영상통화 진행 안내 문자를 보내 드립니다.
  • 영상통화 시 본인명의 휴대폰과 본인 신분증(주민등록증 또는 운전면허증)이 반드시 필요합니다.

  • 영상통화 완료 후 이어서 가입을 진행할 수 있습니다.
  • 예약 가능한 시간이 없을 경우 내일 다시 가입을 신청해 주세요.
