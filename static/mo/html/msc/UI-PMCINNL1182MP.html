보험료 합산이체제도 안내

보험료 합산이체제도란?

  • 자동이체 등록된 계약 중 출금계좌와 이체일이 같은 2건 이상의 계약은 출금거래를 간소화하고자 보험료를 합산하여 이체하는 제도입니다.

보험료 합산이체 방법

  • 자동이체 출금계좌와 이체일이 같은 보험 계약 건은 보험료가 합산되어 해당 이체일에 이체됩니다.
  • 해당 이체일에 자동이체되지 않은 당월 보험료는 5일 단위로 5일, 10일, 15일, 20일, 25일에 다시 청구되며, 이때 보험료가 합산되어 이체됩니다.
  • 단, 납입의무기간이 경과한 유니버셜 계약은 5일 단위로 합산하여 이체되지 않고 건별 이체됩니다.(매월 자동이체 희망일 당일에만 합산 이체됨)
  • 25일까지 출금되지 않은 당월 보험료는 해당월 말일에 합산하여 이체되지 않고 건별 이체됩니다.

보험료 합산이체 예시

  • 김교보 님의 보험계약이 총 9건이고 모두 자동이체 등록된 것을 가정한 예시입니다.

[자동이체 등록현황]

  • 5일자: 2건
  • 10일자: 3건(1건은 납입의무기간이 경과된 유니버셜 상품)
  • 15일자: 1건
  • 말일자: 3건

[3월 5일]

  • 5일 이체 시: 5일자 2건 합산이체

[3월 10일]

  • 5일 이체 시: 10일자 3건 합산이체
  • 5일 미이체 시: 5일자 2건 + 10일자 3건 = 총 5건 합산이체

[3월 15일]

  • 5일, 10일 모두 이체 시: 15일자 1건 이체
  • 5일, 10일 모두 미이체 시
    • ① 5일자 2건 + 10일자 2건(유니버셜 납입의무기간 경과 계약 1건 제외) + 15일자 1건 = 총 5건 합산이체
    • ② 유니버셜 납입의무기간 경과 계약 1건 건별 이체

[3월 20일/25일]

  • 5일, 10일, 15일 모두 미이체 시
    • ① 5일자 2건 + 10일자 2건(유니버셜 납입의무기간 경과 계약 1건 제외) + 15일자 1건 = 총 5건 합산이체
    • ② 유니버셜 납입의무기간 경과 계약 1건 건별 이체

[3월 말일]

  • 25일 모두 이체 시: 말일자 3건 합산이체
  • 25일 모두 미이체 시
    • ① 25일자 미이체 5건 건별 이체
    • ② 유니버셜 납입의무기간 경과 계약 1건 건별 이체
    • ③ 말일자 3건 합산이체

[통장 표기 예시]

표기 내용: 교보005건 또는 교보005G
(G는 ‘건’이라는 의미임)
