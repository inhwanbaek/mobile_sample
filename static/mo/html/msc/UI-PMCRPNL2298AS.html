필요서류를 모두 제출해 주세요.

신청상세사유: 개인회생

  • 신분증
    (주민등록증 또는 운전면허증)
  • 개인회생절차개시결정문 사본
    • 신청일부터 역산(날짜를 뒤에서 앞으로 계산)하여 5년 이내 개시결정 받은 경우 신청 가능
    • 폐지결정 또는 면책결정 시 신청 불가
  • 기타 추가서류
꼭 알아두세요
  • 중도인출신청 접수 시간은 평일 09:00~16:00입니다.(토,일, 공휴일 제외)
  • 기업 관리자가 최종결재를 하고, 필요서류를 모두 제출하면 교보생명에 접수됩니다.
  • 제출한 서류의 유효기간은 교보생명 서류접수일을 기준으로 역산(날짜를 뒤에서 앞으로 계산)하여 적용됩니다.
  • 자세한 사항은 법인 콜센터(1588-0770)로 문의해 주세요.
