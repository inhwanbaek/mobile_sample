대장구분/대장종류 안내

건축물대장은 일반건축물대장과 집합건축물대장으로 나뉩니다. 아래 안내를 참고하여 신청하시려는 지면 건축물의 구분과 종류를 확인하시기 바랍니다.

1. 대장구분

  • 일반: 건물 소유주가 1명이거나 ‘홍길동 외 0명’ 식으로 대표자가 있는 경우
  • 집합: 건물 소유주가 여러 명이며 소유주 이름이 각각 명시되는 경우

일반적으로 아파트, 연립주택, 다세대주택 등은 집합건축물에 해당됨.

2. 대장 종류

(1) 대장구분에서 ‘일반’ 선택 시

  • 총괄 : 신청 주소지 지번의 모든 건물 표시(해당 지번에 건물이 2개동 이상 있을 경우)
  • 일반 : 본인 건물만 표시

(2) 대장구분에서 ‘집합’ 선택 시

  • 총괄 : 해당 지번에 있는 모든 건축물 표시
  • 표제부 : 해당 지번에 있는 동 표시
  • 전유부 : 해당 지번에 있는 동의 호수 표시
  • 예1) 월드컵아파트 2002호 2호의 건축물대장을 신청할 경우
  • ‘대장구분: 집합’, ‘대장 종류: 전유부’ 선택
  • 예2) 단독주택의 건축물대장을 신청할 경우 
  • ‘대장구분: 일반’, ‘대장종류: 일반’ 선택

‘해당되는 정보가 있지 않습니다.’라는 메시지가 나타난다면, 해당 건축물의 주소가 정확하지 않거나, 대장구분 또는 대장종류가 잘못 선택된 경우이니 확인 후 다시 신청하시기 바랍니다.
