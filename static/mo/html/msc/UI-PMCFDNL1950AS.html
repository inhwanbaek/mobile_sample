꼭 알아두세요
  • 설정 시점의 투자수익률보다 상승목표 수익률은 높게 설정하시고, 하락감내 수익률은 낮게 설정하시기 바랍니다.
    예) 현재 투자수익률 20%인 경우,
    → 상승목표 수익률은 20% 이상 입력
    → 하락감내 수익률은 20% 이하 입력
  • 수익률은 소수점 이하는 입력하지 않습니다.
  • 결산으로 인해 당일 분배금이 발생하는 계좌는 실제 수익률과 다를 수 있으니 참고하시기 바랍니다.
  • 목표수익률 알림 등록 시, 문자메시지 수신동의 여부와 관계없이 해당 회차의 상승목표 또는 하락감내 수익률에 도달하면 최초 1회만 최대값을 기준으로 알림이 발송됩니다.
    예) 상승목표 수익률 10%,15% 알림설정 후
    → 투자수익률 9%에서 17% 달성 시 최대값 15% 달성을 기준으로 문자메시지 발송
  • 휴대폰번호가 등록되어 있지 않거나 변경된 경우 [개인정보 > 내정보설정 > 개인정보조회/변경]에서 수정한 후 알림을 설정하시기 바랍니다.
