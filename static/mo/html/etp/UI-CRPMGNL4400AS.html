중도인출 가입자 모바일 신청 설정

꼭 알아두세요
  • 중도인출 가입자 모바일 신청이란?
    • 근로자(가입자)가 직접 필요서류를 첨부해 중도인출을 신청할 수 있는 서비스입니다.
    • 단, 반드시 회사(기업관리자)가 결재해야 교보생명에 접수됩니다.
    • 모바일앱으로 신청할 경우 원본 서류는 제출하지 않아도 됩니다.
  • 퇴직금 중도인출 신청 절차
    • 근로자(가입자)가 교보생명 모바일앱에 로그인
    • [중도인출신청] 메뉴 접속
    • 기본정보 입력 및 사유별 필요서류 첨부
    • 회사(기업관리자)에 결재요청
    • 회사(기업관리자) 결재완료 시 교보생명에서 접수 처리
