1. 부담금 납입대상 선택
4 1

부담금 납입대상을 선택해 주세요.

정기부담금 납입대상이 없습니다.

귀사의 퇴직연금 규약상 납입주기에 따라 계산된 다음 부담금 납입일은 2021년 3월 22일입니다. 해당일 15일 전에 납입대상이 자동으로 생성됩니다.

자동 생성 전에 미리 납입하시려면 [정기부담금 미리 납입]을 눌러 주세요.

  • 납입구분
    미납
  • 미납일
    2021.11.30
  • 진행상태
    결재대기
  • 납입예정금액
    1,000,000원

부담수준 확인 서비스

부담수준 확인 서비스는 가입자별 임금총액을 회사납입금 입력금액과 비교하여 부담금 납입금액이 적정한지 검증하는 서비스로, 이 서비스를 신청하면 부담금 입력 시 모든 가입자의 임금총액을 반드시 입력해야 합니다.

꼭 알아두세요
  • 신규 가입자 또는 퇴직자가 있는 경우에는 신규 가입이나 퇴직 처리를 먼저 해야 합니다.
  • 신규 가입자를 추가하면 추가 즉시 정기부담금 납입대상 목록에 추가됩니다.(미납부담금은 예외)
  • 가입자가 30명 이상인 경우 모바일앱, 모바일웹에서는 부담금 납입을 할 수 없습니다.
