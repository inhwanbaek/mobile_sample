부담수준 확인 서비스

교보생명은 귀사가 관련 법률 「근로자퇴직급여보장법」을 기준으로 적정한 부담금을 산출하여 납입하실 수 있도록 부담수준 확인 서비스를 제공하고 있습니다.

부담수준 확인 서비스는 가입자별 임금총액을 회사납입금 입력금액과 비교하여 부담금 납입금액이 적정한지 검증하는 서비스로, 부담금 입력 시 모든 가입자의 임금총액주1)을 반드시 입력해야 합니다.

주1)
임금총액: 퇴직연금 부담금 산출 시 기준이 되는 금액으로, 규약에서 정한 납입방법인 월납(월임금), 3월납(3개월 임금), 6월납(6개월 임금), 연납(연 임금) 주기에 따라 산정된 임금.
