인증서 가져오기

프로그램 화면에 표시된 인증번호를
입력하시고, 다음 버튼을 터치해 주세요.

  • PC
  • 휴대폰
인증번호

PC에 저장된 공동인증서를 휴대폰으로 복사하면 PC와 모바일에서 동일한 인증서로 모든 금융거래가 가능합니다.

  1. 1PC 홈페이지에서 [PC → 휴대폰 복사] 버튼을 누르면 인증서 목록이 나옵니다.
  2. 2복사할 인증서를 선택하고 인증서 비밀번호를 입력해 주세요.
  3. 3교보생명 앱 실행 후 [인증센터 > 공동인증서 > 인증서 복사 > PC → 휴대폰 복사] 메뉴를 선택해 주세요.
  4. 4PC화면에서 생성된 인증번호를 교보생명 앱 화면에 입력해 주세요.
  5. 5선택하신 공동인증서가 교보생명 앱으로 복사됩니다.
